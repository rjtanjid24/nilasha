# নীলাশা সাহিত্য ই-ম্যাগাজিন — অফিশিয়াল ওয়েবসাইট পোর্টাল
### Nilasha Bengali E-Magazine Official Static Web Portal

একটি আধুনিক, মার্জিত এবং দ্রুত-লোডিং সম্পূর্ণ রেসপনসিভ সাহিত্য সাময়িকী পোর্টাল। এটি **GitHub Pages**-এ হোস্টিং করার জন্য এবং পরবর্তীতে কাস্টম ডোমেইন `https://nilasha.pro.bd` এর সাথে যুক্ত করার উপযোগী করে তৈরি করা হয়েছে। 

এই পোর্টালটি তৈরি করতে **React 19**, **Vite**, **Tailwind CSS v4** এবং **Vanilla-style State Management** ব্যবহার করা হয়েছে, যা একই সাথে প্রিমিয়াম ভিজ্যুয়াল এক্সপেরিয়েন্স এবং দ্রুত লোডিং স্পিড নিশ্চিত করে।

## 📂 প্রোজেক্টের ফোল্ডার স্ট্রাকচার (Folder Structure)

```text
├── /.github/workflows/
│   └── deploy.yml          # স্বয়ংক্রিয় GitHub Actions ডেপ্লয়মেন্ট স্ক্রিপ্ট (CI/CD)
├── /public/
│   ├── 404.html            # কাস্টম ৪MD৪ পেজ (GitHub Pages ফ্রেন্ডলি)
│   ├── CNAME               # কাস্টম ডোমেইন ফাইল (যা nilasha.pro.bd নির্দেশ করে)
│   ├── manifest.json       # PWA কনফিগারেশন ফাইল (অ্যান্ড্রয়েড/আইফোনে ইনস্টল করার জন্য)
│   ├── robots.txt          # সার্চ ইঞ্জিন ক্রলার নির্দেশিকা
│   └── sw.js               # অফলাইন ক্যাশিং সার্ভিস ওয়ার্কার (Offline Cache)
├── /src/
│   ├── App.tsx             # মূল ওয়েবসাইট ফ্রন্টএন্ড কোড এবং ইন্টারঅ্যাকশন লজিক
│   ├── data.ts             # মূল ডেটা সেন্টার (এডিটর, প্রচ্ছদ, সংখ্যা, নিয়মাবলী) [সহজে পরিবর্তনযোগ্য]
│   ├── types.ts            # টাইপস্ক্রিপ্ট ইন্টারফেস ও স্কিমা ডিফিনিশন
│   ├── index.css           # গ্লোবাল সিএসএস, গুগল ফন্টস এবং টেইলউইন্ড থিম ভ্যারিয়েবল
│   └── main.tsx            # রিয়্যাক্ট মাউন্টিং এবং সার্ভিস ওয়ার্কার রেজিস্ট্রেশন
├── index.html              # বেসিক মেটা ট্যাগ, এসইও (SEO), ফেসবুক ওপেনগ্রাফ ও ট্র্যাকিং স্ক্রিপ্ট
├── package.json            # নির্ভরতা লাইব্রেরি ও বিল্ড কমান্ডসমূহ
└── vite.config.ts          # ভিট-কম্পাইলার কনফিগারেশন
```

## 🚀 GitHub Pages-এ ডেপ্লয় করার সহজ ধাপসমূহ (Deployment Guide)

আপনার ওয়েবসাইটটি সম্পূর্ণ ফ্রিতে গিটহাবে লাইভ করার জন্য নিচের সহজ ধাপগুলো অনুসরণ করুন:

1. **GitHub-এ একটি নতুন রিপোজিটরি খুলুন:**
   - আপনার গিটহাব অ্যাকাউন্টে লগইন করে একটি নতুন রিপোজিটরি তৈরি করুন (নাম দিন যেমন: `nilasha-portal`)।

2. **কোড রিপোজিটরিতে পুশ করুন:**
   - আপনার কম্পিউটারে লোকাল ফোল্ডারে গিট চালু করে এই কোডগুলো রিপোজিটরিতে পুশ করুন:
     ```bash
     git init
     git add .
     git commit -m "Initial commit of Nilasha e-magazine portal"
     git branch -M main
     git remote add origin https://github.com/YOUR_USERNAME/nilasha-portal.git
     git push -u origin main
     ```

3. **স্বয়ংক্রিয় বিল্ড ও ডেপ্লয়মেন্ট চালু করুন (GitHub Actions):**
   - আমরা ইতিমধ্যে একটি ডেপ্লয়মেন্ট ওয়ার্কফ্লো স্ক্রিপ্ট `/.github/workflows/deploy.yml` যুক্ত করেছি।
   - আপনি গিটহাবে কোড পুশ করার সাথে সাথে এই ওয়ার্কফ্লোটি স্বয়ংক্রিয়ভাবে রান হবে, প্রজেক্ট বিল্ড করবে এবং `gh-pages` নামক একটি নতুন ব্রাঞ্চে ওয়েবসাইটটি পাবলিশ করে দেবে!
   - **গুরুত্বপূর্ণ সেটিং:** 
     - গিটহাবে আপনার রিপোজিটরির **Settings** -> **Actions** -> **General**-এ যান।
     - স্ক্রোল করে নিচে **Workflow permissions** অপশনে গিয়ে **Read and write permissions** সিলেক্ট করে সেভ করুন। (এর ফলে স্ক্রিপ্টটি সফলভাবে সাইট আপডেট করতে পারবে)।

4. **GitHub Pages সেটিং সম্পন্ন করুন:**
   - আপনার রিপোজিটরির **Settings** -> **Pages** ট্যাবে যান।
   - **Build and deployment**-এর অধীনে **Source** সিলেক্ট করুন `Deploy from a branch`।
   - **Branch** সিলেক্ট করুন `gh-pages` এবং ফোল্ডার `/ (root)` রেখে সেভ করুন।
   - ১-২ মিনিটের মধ্যে আপনার ওয়েবসাইট লাইভ হয়ে যাবে!

## 🌐 কাস্টম ডোমেইন সংযুক্তকরণ (`nilasha.pro.bd`)

ওয়েবসাইটটিকে সরাসরি `https://nilasha.pro.bd` ডোমেইনে চালু করার জন্য নিচের ধাপগুলো অনুসরণ করুন:

### ধাপ ১: DNS রেকর্ডস সেটআপ (আপনার ডোমেইন প্রোভাইডারের পোর্টালে)
আপনার ডোমেইনের কন্ট্রোল প্যানেলে (যেমন: Namecheap, Diana Host, ইত্যাদি) গিয়ে নিচের রেকর্ডগুলো যোগ করুন:

1. **A Records (গিটহাব সার্ভারের দিকে পয়েন্ট করার জন্য):**
   নিচের ৪টি আইপি অ্যাড্রেসের জন্য ৪টি আলাদা **A Record** তৈরি করুন (Host বা Name দিন `@`):
   - `185.199.108.153`
   - `185.199.109.153`
   - `185.199.110.153`
   - `185.199.111.153`

2. **CNAME Record (www সাবডোমেইনের জন্য):**
   একটি **CNAME** রেকর্ড যোগ করুন:
   - **Host / Name:** `www`
   - **Value / Target:** `YOUR_USERNAME.github.io` (এখানে আপনার গিটহাব ইউজারনেম বসান)

### ধাপ ২: গিটহাব সেটিংস পরিবর্তন
1. রিপোজিটরির **Settings** -> **Pages** এ যান।
2. **Custom domain** ঘরে লিখুন `nilasha.pro.bd` এবং সেভ (Save) বাটনে ক্লিক করুন।
3. নিচে **Enforce HTTPS** চেকবক্সটি অন করে দিন (ডোমেইন ভেরিফাই হতে ২-৩ মিনিট বা সর্বোচ্চ কয়েক ঘণ্টা সময় লাগতে পারে, এরপর এটি সচল হবে)।

## ✍️ কীভাবে কন্টেন্ট আপডেট করবেন? (How to Update Content)

**কোনো রিয়্যাক্ট বা প্রোগ্রামিং জ্ঞান ছাড়াই** আপনি ওয়েবসাইটের সমস্ত তথ্য পরিবর্তন করতে পারবেন! এর জন্য আপনাকে শুধু `/src/data.ts` ফাইলটি এডিট করতে হবে। আপনি সরাসরি গিটহাব পোর্টালে গিয়ে এই ফাইলটি ওপেন করে পেনসিল আইকনে ক্লিক করে এডিট করতে পারেন।

### ১. সংখ্যা আহ্বান নোটিশ পরিবর্তন (Edit Call for Submission)
ফাইলটির `currentCall` অংশটি পরিবর্তন করুন:
```typescript
export const currentCall: CallForSubmission = {
  title: "বর্ষা ও শরৎ সংখ্যা ২০২৬", // সংখ্যার নাম
  theme: "স্মৃতি, বৃষ্টি ও কাশফুল (উন্মুক্ত ও থিমভিত্তিক লেখা)", // থিম
  deadline: "১৫ আগস্ট, ২০২৬", // ডেডলাইন
  status: "OPEN", // লেখা নেওয়া চালু থাকলে "OPEN" নতুবা "CLOSED" লিখুন
  requirements: [
    "গল্প: যেকোনো বিষয়ের উপর আকর্ষণীয় ছোটগল্প (সর্বোচ্চ ১,৫০০ শব্দ)।",
    "কবিতা: যেকোনো ছন্দের সর্বোচ্চ ৩টি কবিতা..."
  ],
  submissionEmail: "hellonilasha@gmail.com",
  submissionPhone: "+8809696403683",
  detailsLink: "https://www.facebook.com/nilasha.net"
};
```

### ২. নতুন ম্যাগাজিন সংখ্যা যুক্ত করা (How to Add a Published Issue)
`publishedIssues` তালিকায় নতুন সংখ্যার ব্লক সবার উপরে কপি-পেস্ট করে বসিয়ে দিন:
```typescript
export const publishedIssues: Issue[] = [
  {
    id: "issue-5", // প্রতি সংখ্যার জন্য আলাদা আইডি দিন (যেমন: issue-5, issue-6)
    issueNo: "২য় বর্ষ, ২য় সংখ্যা", 
    title: "শরৎ সংখ্যা ২০২৬",
    publishDate: "সেপ্টেম্বর ২০২৬",
    description: "শরতের মেঘ, শুভ্রতা আর কাশবনের দোলা নিয়ে এবারের নতুন চমক...",
    coverUrl: "https://images.unsplash.com/photo-...", // প্রচ্ছদ ছবির লিংক
    pdfUrl: "https://drive.google.com/file/d/...", // গুগল ড্রাইভ পিডিএফ লিংক
    facebookUrl: "https://www.facebook.com/...", // ফেসবুক পোস্ট লিংক
    featured: true // এটিকে সাম্প্রতিক সংখ্যা হিসেবে দেখাতে "true" রাখুন এবং আগের সংখ্যাটিরfeatured মান "false" করে দিন
  },
  // আগের সংখ্যাগুলো নিচে থাকবে...
];
```

### ৩. প্রচ্ছদ ও প্রচ্ছদ শিল্পীদের তথ্য পরিবর্তন
- **প্রচ্ছদের প্রসেস:** প্রচ্ছদের ছবিগুলো কোনো ফ্রি ইমেজ শেয়ারিং সাইটে (যেমন Unsplash, Imgur, বা পোস্টইমেজ) আপলোড করে সেই ডিরেক্ট ইমেজ লিংকগুলো `coverUrl` বা `avatarUrl`-এ বসিয়ে দিন।
- **সম্পাদকীয় পর্ষদ এডিট:** `editorialBoard` তালিকার সদস্যদের নাম, পদবী এবং ছবির লিংক সহজেই এডিট করতে পারেন।

## 💻 লোকাল ডেভেলপমেন্ট (Local Development)

আপনি যদি নিজের কম্পিউটারে প্রজেক্ট রান করে পরিবর্তন দেখতে চান:

1. **ডিপেন্ডেন্সি ইন্সটল করুন:**
   ```bash
   npm install
   ```
2. **ডেভেলপমেন্ট সার্ভার চালু করুন:**
   ```bash
   npm run dev
   ```
   (সাইটটি ব্রাউজারে `http://localhost:3000` এ সচল হবে)
3. **প্রোডাকশন বিল্ড তৈরি করুন:**
   ```bash
   npm run build
   ```
   (এর ফলে `/dist` ফোল্ডারে বিল্ড ফাইল তৈরি হবে যা গিটহাব পেজে হোস্ট করার উপযোগী)
